หน้าที่รับผิดชอบ

ระบบจัดการสิทธิ์ผู้ใช้งาน (User Access Management)

จัดการสิทธิ์การเข้าถึงในระบบต่างๆ รวมถึง Google Admin, Microsoft Admin และ SAP B1

หน้าที่รับผิดชอบ:
  • สร้าง แก้ไข ลบ และเพิ่มสิทธิ์การเข้าถึงใน Google Admin
  • สร้าง แก้ไข ลบ และดูแลการใช้งาน Microsoft Admin
  • สร้าง แก้ไข ลบ และเพิ่มสิทธิ์การเข้าถึงใน SAP B1 สำหรับผู้ใช้งาน
  • ดูแลระบบเครือข่าย (Network) ให้มีประสิทธิภาพสูงและพร้อมใช้งานตลอดเวลา
  • จัดทำข้อมูลพนักงานใหม่ รวมทั้งถ่ายรูปและรวบรวมข้อมูลเพื่อจัดทำบัตรพนักงานใหม่
เทคโนโลยีที่ใช้:
Google Admin Console Microsoft 365 SAP Business One Active Directory Network Management

โปรเจกต์และผลงาน

ระบบบำรุงรักษาอุปกรณ์ IT (Preventive Maintenance)

ดูแลและบำรุงรักษาอุปกรณ์คอมพิวเตอร์ เครื่องพิมพ์ และเครื่องอ่านลายนิ้วมือ

หน้าที่รับผิดชอบ:
  • ดูแลคอมพิวเตอร์ เครื่องพิมพ์ เครื่องอ่าน/เก็บลายนิ้วมือ และอุปกรณ์อื่นๆ ตามรอบการบำรุงรักษา
  • ติดตั้งระบบปฏิบัติการและลงโปรแกรมให้กับลูกค้า
  • ซ่อมและแก้ไขปัญหา Hardware และ Software ให้กับลูกค้า
  • จัดทำรายงานอุปกรณ์และเอกสาร PM ประจำเดือน
เทคโนโลยีที่ใช้:
Hardware Maintenance Software Updates Biometric Systems Printer Management

ระบบ FTP และ File Transfer Protocol

จัดทำและดูแลระบบ FTP เพื่อการแชร์ไฟล์และข้อมูลภายในองค์กร

หน้าที่รับผิดชอบ:
  • จัดทำระบบ FTP (File Transfer Protocol) ให้กับองค์กร
  • กำหนดสิทธิ์การเข้าถึงและความปลอดภัยของระบบ
  • ดูแลและบำรุงรักษาเซิร์ฟเวอร์ FTP ให้ทำงานได้อย่างต่อเนื่อง
เทคโนโลยีที่ใช้:
FTP Server Network Configuration File Management Security Protocol

ระบบสนับสนุนการขายและ Demo

เตรียมอุปกรณ์และระบบสำหรับการนำเสนอขายให้กับลูกค้า

หน้าที่รับผิดชอบ:
  • สนับสนุนทีมขายในการ Demo ระบบงานต่างๆ สำหรับงานขายร่วมกับทีม Sale
  • ฝึกอบรมการใช้งานระบบให้กับลูกค้า
  • ให้คำปรึกษาและแก้ไขปัญหาทางไกล (Remote) ให้กับลูกค้า
เทคโนโลยีที่ใช้:
System Demo Sales Support Equipment Setup Customer Training Remote Support

ระบบบริการประชาชนเคลื่อนที่

จัดเตรียมอุปกรณ์และเครื่องผลิตบัตรสำหรับสำนักทะเบียน

หน้าที่รับผิดชอบ:
  • จัดเตรียมอุปกรณ์และเครื่องผลิตบัตรให้กับสำนักทะเบียน เพื่อออกหน่วยบริการประชาชนเคลื่อนที่
  • ติดตั้งระบบคอมพิวเตอร์และอุปกรณ์ต่างๆ ให้กับลูกค้า
  • ประสานงานกับหน่วยงานหรือผู้เกี่ยวข้อง (Vendor) เพื่อแจ้งและส่งซ่อมอุปกรณ์
เทคโนโลยีที่ใช้:
Mobile Service Unit Card Printing System Government Service Equipment Management

ร้านซ่อมคอมพิวเตอร์

เปิดร้านรับซ่อมคอมพิวเตอร์ ประกอบเครื่อง และติดตั้ง IP Camera

หน้าที่รับผิดชอบ:
  • รับซ่อมคอมพิวเตอร์ทั้งด้าน Hardware และ Software
  • จัดสเป็คและประกอบคอมพิวเตอร์ตามความต้องการของลูกค้า
  • ติดตั้งระบบปฏิบัติการ (OS) และโปรแกรมต่างๆ
  • ติดตั้งระบบ IP Camera สำหรับบ้านและสำนักงาน
  • ให้คำปรึกษาด้านเทคโนโลยีแก่ลูกค้า
เทคโนโลยีที่ใช้:
Hardware Repair PC Assembly OS Installation IP Camera Installation Customer Service
